# Labs

권장 구조:

```text
labs/
  ch01_data_quality/
  ch02_model_quality/
  ch03_serving/
  ch04_observability/
  ch05_qa_strategy/
```

서버, Docker, MLflow, Grafana처럼 환경 차이가 큰 항목은 무조건 실행하지 않습니다. 먼저 prepared artifact를 확인하고, 강사가 로컬 실행을 지시한 경우에만 실행합니다.
