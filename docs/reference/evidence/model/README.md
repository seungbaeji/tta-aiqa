# Model Evidence Index

## 1. Historical V1

### 1-1. Scenario review

The files directly under this directory are V1 historical evidence. V1 held both
candidates and remains preserved as a scenario-review record. It must not be
used to determine the current course deployment.

## 2. Current Course Revision

### 2-1. V2 approval

The current course scenario is
[`revisions/v2/`](revisions/v2/). Its canonical result is Candidate A
`HOLD`, Candidate B `APPROVE`, and deployment is allowed only through
`revisions/v2/release-manifest.json`.

See [the V2 evidence guide](revisions/v2/README.md) for the ordered
train/valid, sealed-test, and publication records.
